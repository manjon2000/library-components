# Radio Button
Use radio buttons when you have a group of mutually exclusive choices and only one selection from the group is allowed.

## When to use

### Form
Can be used in tiles, data tables, modals, side panels, and in forms on full pages.

### Settings
Used to change from one setting to another in a menu, page, or component. It can often act as a filtering mechanism.


### Customize component
To give custom styles using CSS variables, we have the following variables:


For sizes it is recommended to use `em` units.
```CSS
.custom-radiobutton {
  --ui-radiobutton-border-default: #000;
  --ui-radiobutton-border-focus: green;
  --ui-radiobutton-border-checked: red;
  --ui-radiobutton-shadow-bg-checked: blue;
  --ui-radiobutton-border-error: red;
  --ui-radiobutton-shadow-bg-default: transparent;
  --ui-radiobutton-shadow-bg-checked: violet;
  --ui-radiobutton-shadow-bg-error: red;
  --ui-radiobutton-shadow-bg-disabled: transparent;
  --ui-radiobutton-shadow-width-default: 0px;
  --ui-radiobutton-shadow-width-checked: 5px;
  --ui-radiobutton-focus-outline-width: 1px;
  --ui-radiobutton-focus-outline-style: solid;
  --ui-radiobutton-focus-outline-color: red;
  --ui-radiobutton-focus-outline-offset: 2px;
}
```
#### Use case custom component

```html
<ui-radiobutton
  class="custom-radiobutton"
  [label]="label"
  [isChecked]="isChecked"
  [isDisabled]="isDisabled"
  [isError]="isError"
/>
```
